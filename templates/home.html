This server provides two methods for obtaining culled PDB subsets.

  1. Subsets of the PDB can be culled by pairwise sequence identity and structure quality. The subsets can be defined by the user uploading a set of PDB chains/entries, or by the user selecting to cull the proteome of a given organism.
  2. Alternatively, pre-computed sets of non-redundant PDB chains can be downloaded.


For user that wish to cull arbitrary FASTA files, the source code for the Leaf algorithm can be downloaded.

Contact information for the website and authors of the paper can be found here.

Information about the extraction of the PDB data stored by the server can be found here.

Further information about the algorithm can be found in the paper describing it.
