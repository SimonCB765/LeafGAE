This page provides downloads for non-redundant datasets, generated using a variety of structural criteria.
CHANGE THE WORDING
The name of each download follows a defined format. The format is: SeqIden_X_Res_Y_RVal_Z. X is the percentage sequence identity threshold used. This means that if X is 20, then there are no two sequences in the non-redundant dataset with a pairwise sequence identity greater than 20%. Y is the maximum resolution of the protein structure. Z is the maximum R value. If the name of the download ends in INCLNONXRAY_INCLCAONLY, then the dataset from which the redundancy was removed contained non-X-ray structures and chains where the structure contains only backbone alpha carbon atoms.

The non-redundant dataset is recorded in a FASTA file, with the same format as the FASTA files returned by the user generated redundancy removal.
